# 🪟 Leo Vidros — Digitalização & Sistema de Gestão

> Projeto incentivado pela **FIAP São Paulo Tech School** em parceria com a comunidade.

## Sobre o Projeto

A **Leo Vidros** atua no ramo de vidraçaria há mais de 5 anos e está passando 
pelo processo de digitalização. Este projeto nasce da parceria com a SPTECH para 
solucionar as principais dores operacionais da empresa por meio de um sistema 
moderno e completo.

## Solução

- ☁️ **Infraestrutura AWS** — Ambiente cloud robusto e escalável
- 📦 **Gestão de Estoque** — Controle de materiais em tempo real
- 📋 **Pedidos & Agendamentos** — Fluxo completo de atendimento ao cliente
- 👥 **Gestão de Funcionários** — Cadastro e controle de equipe
- 📄 **Geração de Orçamentos em PDF** — Documentos profissionais automáticos
